This help article describes the properties belonging to the T:Telerik.UI.Chart._AxisLabelDateFormatConfiguration
				class, which are available when setting the axes' labels date format. The format that will be used depends on the axis __
					baseUnit
				__ value. For example, if the base unit is set to "years" the format provided by axis.labels.dateFormats.years will be
				applied.
			

# Section1Axis Label Date Format Configuration

Name

Description

__days__

Sets the format for displaying day values in the axis labels. Default value is "M/d".

__hours__

Sets the format for displaying hour values in the axis labels. Default value is "HH:mm".

__months__

Sets the format for displaying month values in the axis labels. Default value is "MMM 'yy".

__years__

Sets the format for displaying year values in the axis labels. Default value is "yyyy".
